# BD Clínica de Psicologia

Banco de dados desenvolvido para gerenciamento de uma clínica de psicologia.

## Funcionalidades
- Cadastro de pacientes
- Cadastro de psicólogos
- Agendamentos
- Consultas
- Controle administrativo

## Tecnologias
- MySQL
- SQL

## Como usar
1. Baixe o arquivo `clinica_psicologia.sql`
2. Abra o phpMyAdmin ou MySQL Workbench
3. Importe o arquivo
4. Execute o script
