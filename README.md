# BD Clínica de Psicologia

Banco de dados desenvolvido para gerenciamento de uma clínica de psicologia.

## Funcionalidades
- Cadastro de pacientes e psicólogos
- Agendamentos
- Consultas
- Controle administrativo

## Tecnologias
- MySQL
- SQL
- JavaScript, CSS e HTML
